# Git Commands for Beginners

This repository is my personal learning notebook for Git and GitHub.
The goal is to understand git basics and practice them daily.

This plan is suitable for beginners and students.


## Why this repo?
- To practice Git daily
- To get comfortable with GitHub
- To help other beginners



##  Day 1 – Git Setup
- Learn what Git is
- Configure username and email
- Understand why setup is needed

Commands:
- git config
- git config --global --list

---

##  Day 2 – Initialize Repository
- Learn how Git starts tracking a project
- Create a local Git repository

Commands:
- git init
- git status

---

##  Day 3 – Staging Files
- Understand working directory and staging area
- Add files for commit

Commands:
- git add
- git reset
- git status

---

##  Day 4 – Commit Changes
- Learn what a commit is
- Save changes properly with messages

Commands:
- git commit -m
- git log

---

##  Day 5 – View Changes
- Check what is changed
- Compare file versions

Commands:
- git diff
- git diff --staged

---

##  Day 6 – Branch Basics
- Understand what branches are
- Create and switch branches

Commands:
- git branch
- git checkout

---

##  Day 7 – Merge Branches
- Combine work from different branches
- Learn safe merging

Commands:
- git merge

---

##  Day 8 – Connect to GitHub
- Learn what a remote repository is
- Connect local project to GitHub

Commands:
- git remote add
- git remote -v

---

##  Day 9 – Push & Pull
- Send code to GitHub
- Get updates from GitHub

Commands:
- git push
- git pull
- git fetch

---

##  Day 10 – Practice Workflow
- Practice full Git workflow
- Make changes, commit, push
- Review commit history

Commands:
- git status
- git add
- git commit
- git push
- git log

---

##  Goal of This Plan
- Get comfortable with Git commands
- Understand how GitHub works
- Build confidence using Git in projects


This repository is made for learning and practice.
Mistakes are part of the learning process.
